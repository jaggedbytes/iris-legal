## Enable GitHub Pages

1. Open the repository on GitHub → **Settings** → **Pages**
2. **Source:** Deploy from a branch
3. Branch: `main` / folder: `/ (root)`
4. Save, then wait a minute for the site to publish

## A2P message-flow snippet

Use this with the live privacy and terms URLs:

> Trusted contacts are enrolled by the Iris account holder or an authorized operator in the Iris dashboard, where the contact’s name and mobile number are provided for care-related check-in messages.
